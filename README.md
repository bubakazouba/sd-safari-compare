# SD Zoo Safari Park — Safari Comparison

A one-page dashboard comparing every bookable safari at the [San Diego Zoo Safari Park](https://sdzsafaripark.org/safaris).

## Live page

→ **https://bubakazouba.github.io/sd-safari-compare/**

## What's in here

- `index.html` — sortable, filterable comparison table (vanilla JS, no build)
- Filter for **mobility-friendly** options (vehicle-based, minimal walking)
- Filter to hide overnight options if you only want day trips
- Click any column header to sort

## Data

All pricing and durations were scraped from `sdzsafaripark.org/safaris` on 2026-04-19. Prices are *per person, plus admission* (admission is ~$72 adult separately). Verify on the official site before booking.

## Categories

| Mobility tier | What it means |
|---|---|
| 🟢 Vehicle / Customizable | Truck or private cart, no walking required |
| 🟡 Cart + walk | Cart-based but with disembark stops on uneven terrain |
| 🔴 Walk / stairs | Significant walking required (or zipline stairs) |
